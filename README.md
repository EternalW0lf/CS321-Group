# Restaurant Budget Optimizer (RBO)

## Overview
A web-based MVP that allows restaurant managers to upload or manually input data, set profit goals, and analyze menu performance through data-driven insights.

## Features (Sprint 1 MVP)
- Upload CSV data (menu + customer stats)
- Input target profit percentage
- View basic results dashboard
- Login / Signup UI

## Features (Sprint 2 MVP)
- Dynamic results dashboard (overview, per-item insights, trends)
- Manual data entry for menu items, ingredients, and customer data
- CSV validation and format enforcement
- Login / Signup with localStorage-based session handling
- User-specific analysis history (save, reload, delete, and name analyses)
- Result export (download functionality)
- Enhanced feedback and ingredient breakdown per menu item

## How to Run
Download the repository and open `index.html` in a web browser.

## Tech Stack
- HTML
- CSS
- JavaScript
- Papa Parse (CSV parsing)
- localStorage (client-side persistence)
